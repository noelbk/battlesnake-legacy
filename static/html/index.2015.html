Frequently Asked Questions

What is BattleSnake?

BattleSnake is a programming competition like no other. Write your snake AI and battle to the death against other teams. Only one snake will emerge victorious.

Do I need to code?

No, but at least one person on your team will have to. Teams can include first time programmers, designers or anyone who likes playing games.

What programming is required?

Teams will create a simple web server that plays the game using HTTP requests. Each AI will receive information about the game and then respond with its next move.

How do I register?

Please register individually or as a team on our registration page.

I have more questions. Who should I ask?

That's easy! Shoot Linley an email at battlesnake@sendwithus.com.

What are the game rules? How does it work?

Don't worry about it. You'll know when the time comes. To keep things fair, we will go over game rules on stage at the launch of the event. Don't be late.

What technologies will we use?

All snake AIs will deploy using Heroku and play the game using an HTTP API. You may use the programming language of your choice. We will provide additional docs for Python, Java, Node, and Go.

How should I prepare for battle?

Participants are strongly encouraged to bring a laptop capable of writing and deploying code to Heroku using Git and GitHub. To get started with Heroku, see this guide.

Can I go alone?

Yes. If you dare...
